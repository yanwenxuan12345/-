# 摄影接单网站 - 导出与部署指南

## 项目简介
这是一个专业的摄影接单网站，包含服务展示、作品集、在线预约等功能。

## 如何导出/构建项目

### 前置条件
- 安装Node.js (v14.0.0或更高版本)
- 安装pnpm包管理器

### 构建步骤

1. **安装依赖**
   ```bash
   pnpm install
   ```

2. **构建生产版本**
   ```bash
   pnpm build
   ```

3. **构建结果**
   构建完成后，会在项目根目录生成`dist`文件夹，包含所有可部署的静态文件。

## 导出文件说明
`dist`文件夹包含以下内容：
- HTML文件
- CSS样式文件
- JavaScript脚本
- 图片资源
- 其他静态资源

## 部署指南
构建完成的`dist`文件夹可以部署到任何静态网站托管服务，如：
- Netlify
- Vercel
- GitHub Pages
- 阿里云OSS
- 腾讯云COS

只需将`dist`文件夹中的所有文件上传到您选择的托管服务即可。

## 开发模式
如果需要继续开发，可以使用开发服务器：
```bash
pnpm dev
```
